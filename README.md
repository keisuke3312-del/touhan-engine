# 登録販売者 過去問中心問題生成エンジン v0.3

GitHub Pages用の無料運用版です。

## v0.3で追加
- 解答・解説の表示/非表示
- 間違えた問題だけ復習表示
- 問題検索
- 学習履歴バックアップJSON保存
- 学習履歴バックアップJSON貼り付け復元

## 既存機能
- 第1〜第4セット、計120問生成
- 表示セットごとのJSONコピー・保存
- 4セット統合JSON保存
- 答えなし問題表示
- 回答文字列による採点
- 採点JSONコピー・保存
- localStorageによる累計・章別正答率保存
- 誤答履歴を次回生成へ軽く反映

## 無料運用方針
- GitHub Pages無料枠
- 有料APIなし
- 有料DBなし
- 有料サーバーなし
- 保存はlocalStorage又は手動JSONバックアップ
