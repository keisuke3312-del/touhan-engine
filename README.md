# 登録販売者 過去問中心問題生成エンジン v0.2

GitHub Pages用の最小構成です。

## 使い方
1. このリポジトリをGitHub Pagesで公開
2. SafariでURLを開く
3. iPhoneの共有メニューから「ホーム画面に追加」

## 現在できること
- 第1〜第4セット、計120問生成
- 表示セットごとのJSONコピー・保存
- 4セット統合JSON保存
- 答えなし問題表示
- 回答文字列による採点
- 採点JSONコピー・保存
- localStorageによる累計・章別正答率保存
- 誤答履歴を次回生成へ軽く反映

## 無料運用方針
- GitHub Pages無料枠
- サーバーなし
- 有料APIなし
- 有料DBなし
- 保存はlocalStorage又はJSON手動保存

## 注意
Publicリポジトリに置いた内容は第三者にも閲覧可能です。
公開して問題ないデータだけ配置してください。
