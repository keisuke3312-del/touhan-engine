登録販売者 過去問エンジン v1.0.0

一問一答で、理由節だけが残った断片文（「〜ため。」「〜ので。」等）を生成候補から除外。

# 登録販売者 過去問エンジン v1.0.0

- 東京2021〜2025年度の過去問DBを検査
- 本番形式：30問×4セット（120問）生成
- 一問一答：公式問題の各記述を○×へ分解し、30問×4セット生成
- 学習アプリ互換JSONを保存
- 章配分は原則 5・5・5・10・5問／セット


## v1.0.0
- 一問一答120問内で、同じ知識を言い換えただけの類似問題を自動除外
- 文字列完全一致だけでなく、3文字単位の類似度・包含率で判定
- 「本剤」「本製剤」「前記」「上記」「このうち」など文脈依存表現の除外を強化
- 類似問題除外後も章配分を維持し、不足時は別候補へ差し替え

- 正誤表の列見出し `a b c d` を本文記述として誤認しないよう修正
- 列見出しが2行に分断された場合も除外
- a〜d欠落問題を生成候補から除外
- 2025年度の正誤表形式を重点修正

## v1.0.0
- 一問一答で「本剤」「本品」など参照先がない文脈依存問題を除外
- 「陽性界面活性」などOCR崩れを含む不自然な記述を除外
- 「作用を真菌類示す」など語順破綻した記述を除外


## v1.0.0
- 問題JSONへ正答・短い解説・詳しい解説を統合
- 採点用解答JSONの個別出力を廃止
- 旧一問一答画面を削除
